## Key Outcomes
1. Prisma schema bổ sung enum `ZaloAccountPermissionLevel` + model `ZaloAccountAssignment` và relation vào `ZaloAccount`.
2. Migration mới tạo bảng assignment với index phục vụ query và unique active assignment theo (`tenant_Id`,`zaloAccountId`,`userId`) khi `revokedAt IS NULL`.
3. Tạo `ZaloAccountAssignmentService` để:
   - resolve account access theo auth context,
   - assert `READ/CHAT/ADMIN`,
   - list/upsert/revoke assignment,
   - resolve accessible account ids cho filter conversations.
4. API assignment mới trong Zalo controller/service:
   - `GET /api/v1/zalo/accounts/:id/assignments`
   - `PUT /api/v1/zalo/accounts/:id/assignments/:userId`
   - `DELETE /api/v1/zalo/accounts/:id/assignments/:userId`
5. Cắm enforce quyền:
   - `sendPersonalMessage`/`sendOaMessage` yêu cầu `CHAT`.
   - Conversations list/messages/evaluation lọc/kiểm quyền theo assignment account.
6. Rollout-safe behavior giữ nguyên cho role đặc quyền:
   - `ADMIN`/`MANAGER` bypass assignment cho read/chat,
   - `STAFF` áp assignment scope,
   - quản trị assignment (`list/upsert/revoke`) chỉ cho `ADMIN`.
