2. **Enterprise controls**
   - Audit trail bắt buộc cho update/validate/test-connection/snapshot/restore.
   - Snapshot versioning + restore theo domain.
   - `dryRun` support trong `updateDomain`.
   - `validateDomain` audit được theo request.

3. **Secret strategy (env/secret store only)**
   - Allowlist `secretRef`: `BHTOT_API_KEY`, `AI_OPENAI_COMPAT_API_KEY`, `ZALO_OA_ACCESS_TOKEN`, `MEILI_MASTER_KEY`.
   - Secret resolver runtime qua env (`resolveSecretByRef`).
   - Sanitize plaintext secret fields cho `integrations` trước khi persist.
   - API trả trạng thái cấu hình (`isConfigured`, `hasSecret`) thay vì secret value.

4. **Settings API + backward compatibility**
   - Cập nhật `settings.controller.ts` + `settings.service.ts` với endpoint mới:
     - `GET /settings/center`
     - `GET|PUT /settings/domains/:domain`
     - `POST /settings/domains/:domain/validate`
     - `POST /settings/domains/:domain/test-connection`
     - `GET /settings/audit`
     - `POST /settings/snapshots`
     - `GET /settings/snapshots`
     - `POST /settings/snapshots/:id/restore`
   - Duy trì endpoint cũ:
     - `/settings/config`
     - `/settings/search/*`
     - `/settings/bhtot/*`
   - Bridge dual-read/dual-write key legacy:
     - `system_config`
     - `order_settings`
     - `finance_period_locks`
