## Scope
Fix lỗi không hiển thị ảnh QR ở modal đăng nhập Zalo Automation accounts.

## Diagnosis
1. API `GET /api/v1/zalo/accounts/:id/personal/qr` trả `qrImage` dạng base64 thô (prefix `iVBOR...`).
2. Frontend render trực tiếp chuỗi đó vào `<img src>`.
3. Browser yêu cầu data URL hoặc URL hợp lệ, nên ảnh bị vỡ.

  - thêm `normalizeQrImageSource(...)`.
  - tự prepend `data:image/png;base64,` nếu chuỗi là base64 thô.
  - áp dụng normalize ở 2 đường:
    - socket realtime `zalo:qr`
    - API fallback `/personal/qr`
- Cập nhật regression test:
  - `apps/web/e2e/tests/zalo-automation-accounts-actions.spec.ts`
  - mock `qrImage` base64 thô và assert `img.zalo-qr-image[src^=\"data:image/png;base64,\"]`.
